# life-blog

#### server:
  1. cd server
  2. create folder 'uploads' - this folder will be save photos to posts and avatars to profiles
  3. create folders: 'users' & 'posts' into folder 'uploads'
  4. npm i - install dependencies
  5. npm start <br/><br/>
  
#### client: <br/>
  1. cd client
  2. npm i - install dependencies
  3. npm start
